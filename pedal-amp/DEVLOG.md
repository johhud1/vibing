# Development Log

This project's development progress is tracked in the `docs/` directory:

- **[Tasks & Progress](docs/TASKS.md)**: Detailed checklist of implemented features and pending work.
- **[Release Notes](docs/RELEASE_NOTES.md)**: Walkthrough of changes in the latest build.
- **[Implementation Plan](docs/IMPLEMENTATION_PLAN.md)**: Technical design and implementation details.

## Contributing

To contribute to this project:
1. Check `docs/TASKS.md` for open items.
2. Follow the implementation patterns in `docs/IMPLEMENTATION_PLAN.md`.
3. Submit a PR.
